# Hospital-Emergency-Room-Excel-Dashboard

This project presents an interactive Excel dashboard designed to analyze hospital emergency room data. The dashboard provides insights into patient admission status, waiting time, demographics, and department referrals to support better decision-making in healthcare operations.
Tools & Technologies
Microsoft Excel | Pivot Tables | Pivot Charts | Data Cleaning | Data Analysis | Dashboard Design
Key Insights
* Patient Admission vs Non-Admission Analysis
* Patient Age Group Distribution
* Emergency Room Wait Time Analysis
* Gender Distribution of Patients
* Department Referral Trends
Dashboard Features
* Interactive charts and filters
* KPI-based analysis
* Clear visualization of hospital emergency room performance
* Data-driven insights for operational improvement
